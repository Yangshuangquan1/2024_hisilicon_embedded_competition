# 残障人士避障辅助器-基于Hi3861的盲人避障辅助器


本项目旨在开发一款基于海思 Hi3861的盲人避障辅助器，通过超声波传感器检测前方障碍物，利用语音提示和震动反馈，帮助盲人用户安全避障。

## 功能特点

- 实时检测前方障碍物距离
- 语音提示障碍物方向和距离
- 震动反馈帮助用户感知障碍物
- 简单易用，携带方便

## 使用说明

1. 打开电源，系统开始初始化
2. 系统启动后，超声波传感器开始检测前方障碍物距离
3. 当检测到障碍物时，系统通过语音和震动提示用户
